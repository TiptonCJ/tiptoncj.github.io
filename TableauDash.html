Project Overview

This project is a data dashboard created in Tableau, showcasing the analysis of the ADCC (Abu Dhabi Combat Club) dataset from Kaggle. The dashboard includes various visualizations that provide insights into fighter performance, match statistics, and long-term trends.

Key Insights & Features

  • Over 1000 matches analyzed across divisions, genders, and outcomes.
  • Filter by fighter, gender, weight class, or submission type.
  • Note: Women's divisions began in 2011 and include only two weight classes.
  • Absolute division is open-weight for only available top-performing male athletes in their respective divisions.
