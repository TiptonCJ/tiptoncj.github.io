Project Overview

Diamondtrack is a comprehensive live baseball stats tracking application developed as my capstone project. Inspired by platforms like GameChanger, it allows coaches, players, and fans to track real-time game statistics, manage team rosters, and analyze performance data.

Built with modern web technologies, Diamondtrack provides an intuitive interface for recording plays, viewing live scoreboards, and generating detailed reports.

Key Features

  • Real-time game tracking with live score updates
  • Player and team statistics management
  • Customizable dashboards for coaches and players
  • Data visualization for performance analysis
  • Mobile-responsive design for on-the-go access
  • Integration with common baseball scoring systems

Technologies Used

Diamondtrack was built almost entirely in TypeScript utilizing the Expo Go app and framework. It provides an intuitive interface for recording plays, viewing live scoreboards, and generating detailed reports.
